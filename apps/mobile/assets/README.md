# Assets

Drop the following PNGs here before building:

- `icon.png` — 1024x1024 app icon (warm cream bg, terracotta `ש` logomark)
- `splash.png` — 1242x2436 splash image
- `adaptive-icon.png` — 1024x1024 Android foreground
- `notification-icon.png` — Android notification icon (optional)

During development, Expo will generate placeholders automatically based on the
config, but final submission to the App Store / Play Store requires real assets
matching the design in `/tmp/design/shakana/project/Shakana App.html`
(accent `#c96a3b`, background `#faf8f4`, Playfair Display `ש` glyph).
